# Document Skills

이 skill은 프로젝트의 문서화 작업을 도와줍니다.

## 기능

### 1. README 생성
프로젝트 구조를 분석하여 README.md를 작성합니다.
- 프로젝트 소개, 설치 방법, 사용법, 기여 방법 포함

### 2. 코드 주석 추가
함수, 클래스, 모듈에 명확한 주석을 추가합니다.
- JSDoc, Python docstring, 인라인 주석 등

### 3. API 문서 작성
API 엔드포인트, 파라미터, 응답 형식을 문서화합니다.

### 4. 변경 이력 작성
git 로그를 기반으로 CHANGELOG.md를 생성합니다.

## 사용법

`/document-skills` 명령어 후 원하는 작업을 요청하세요:
- "README 작성해줘"
- "이 함수에 주석 추가해줘"
- "API 문서 만들어줘"
- "CHANGELOG 작성해줘"
